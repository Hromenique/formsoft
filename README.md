Formsoft
========

Sistema de controle de produtos e pedidos.

O sistema possui as seguintes funcionalidades:

* Login com usuário e senha

* Cadastrar Produto

* Cadastrar Pedido

* Relatório:

	Posição Quantitativa e Financeira dos Produtos

	Relatório dos Produtos abaixo do Mínimo

	Relatório de Vendas Efetuadas


Tecnologias utilizadas
----------------------

CDI, JPA/Hibernate, JSF/PrimeFaces, Spring Security, Postgresql, JasperReport, e JUnit.
